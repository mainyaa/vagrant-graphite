# vagnrat-graphite

## Setup Vagrant

```
brew tap phinze/homebrew-cask
brew install brew-cask
brew cask install virtualbox
brew cask install vagrant
vagrant box add lucid32 http://files.vagrantup.com/lucid32.box
```

# Setup Berkshelf
```
gem install berkshelf
berks install
```


## Run it!

```
vagrant up
vagrant ssh
```
